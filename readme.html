nl-discount:compile-markdown

syntax: (nl-discount:compile-markdown input-file output-file [flags])
parameter: input-file - The markdown file you would like compiled to html
parameter: output-file - The resulting file after compilation
parameter: flags - - optional; flags for modifying the output

return: Returns nil



The function compile-markdown uses libc's fopen/fclose calls in conjunction with mkd_in / markdown / mkd_cleanup to compile a file into a html output. (abstracting away from directly dealing with the C library)

example:
 (nl-discount:compile-markdown "test.md" "out.html" (| TOC NOSTRIKETHROUGH))
 => nil

nl-discount:markdown-string

syntax: (nl-discount:markdown-string input-string [flags])
parameter: input-string - The string containing markdown you would like markdown'd
parameter: flags - - optional; flags for modifying the output

return: Returns nil



The function markdown-string compiles a string of markdown valid syntax and returns it as a string

example:
 (nl-discount:markdown-string "*oh hai*" (| TOC NOSTRIKETHROUGH))
 => "<p><em>Oh hai</em></p>"
